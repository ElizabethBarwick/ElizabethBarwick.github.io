Data Cleaning with SQL:
Nashville Housing

This project involved a dataset of housing sales from Nashville. I identified duplicates and removed them, split address strings into separate parts to make them more usable, replaced nulls by identifying similar entries, added new columns created from existing data, standardized formats across dates and eliminated inconsistencies in data entry. This work was completed as a guided project.

Data Exploration with Python:
Home Equity Loan Defaults

This project involved extensive data exploration. The data set contains loan and borrower characteristics for 5960 loans. I performed univariate analysis, bivariate analysis and multivariate analysis. Libraries used include Numpy, Pandas, Matplotlib and Seaborn. I created countplots, histograms, boxplots, pairplots and a heatmap to better understand the relationships among the variables.

Machine Learning:
Loan Default Prediction

Given a historical dataset including both loan and borrower characteristics, I trained various classication models on the data in order to optimize prediction of loan defaults for home equity loans. Loan defaults result in significant loss of profits for banks. The goal was to maximize recall, or the ability to correctly classify loans that actually defaulted, while at the same time maintaining high precision so as not to turn away eligible borrowers. Models trained and tuned included logistic regression, decision tree and random forest. Additional tuning was performed using Adaboost, XGBoost and Gradient Boost. The best performing model was the decision tree with a recall of 82.5% and a precision of 93.5%.
